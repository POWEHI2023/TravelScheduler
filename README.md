# TravelScheduler

## 简介

TravelScheduler 是一个基于 SwiftUI 和 MapKit 的 iPhone / iPad 行程路线规划应用，适合用于旅行路线整理和多地点出行安排。应用支持将多个地点组织为一条可视化路线，并结合地图展示、分段规划和路线摘要，帮助用户更直观地安排出行。

## 功能

- 搜索景点、地标或地址，并将地点加入行程。
- 支持对已选地点进行删除、重排和顺序调整。
- 支持设置起点、终点，以及终点回到起点的环线模式。
- 支持为每一段路线选择 `driving`、`walking`、`transit` 三种通行方式。
- 在地图中按分段展示路线，并支持逐段隐藏或显示。
- 查看路线总时长、总距离、分段详情和整体出行建议。
- 生成 Markdown 格式的路线规划文档，并可在应用内复制内容。

## 运行方式

1. 使用 Xcode 打开 `TravelScheduler.xcodeproj`。
2. 选择 `TravelScheduler` target。
3. 选择 iPhone 或 iPad 模拟器，或连接真机后运行项目。
